## 1. Отправка fuzz-запроса и фиксация ответа

Файл `backend/fuzzing/run.ts`. Универсальная функция HTTP-запроса с замером времени; поддерживает JSON-тело и «битый» rawBody для проверки парсера.

```typescript
async function request(method: string, path: string, options = {}) {
  const headers = { ...options.headers };
  if (options.rawBody !== undefined) {
    headers['Content-Type'] = 'application/json';
  } else if (options.body !== undefined) {
    headers['Content-Type'] = 'application/json';
  }
  if (options.token) {
    headers.Authorization = `Bearer ${options.token}`;
  }
  const start = performance.now();
  const res = await fetch(`${BASE_URL}${path}`, {
    method,
    headers,
    body: options.rawBody ?? (options.body !== undefined ? JSON.stringify(options.body) : undefined),
  });
  const text = await res.text();
  return { status: res.status, text, durationMs: Math.round(performance.now() - start) };
}
```

## 4. Валидация на сервере (Zod)

Файл `backend/src/middleware/validateRequest.ts`. Проверка до вызова контроллера; при ошибке — 400 через errorHandler.

```typescript
export function validateRequest(schemas: RequestSchemas) {
  return (req, _res, next) => {
    if (schemas.body) req.body = schemas.body.parse(req.body);
    if (schemas.params) req.params = schemas.params.parse(req.params);
    next();
  };
}
```

## 5. Доработка после fuzz: безопасные строки

Файл `backend/src/validators/safeString.ts`. Устраняет 500 при NUL-байте в полях, передаваемых в PostgreSQL.

```typescript
export function safeString(min: number, max: number) {
  return z
    .string()
    .min(min)
    .max(max)
    .refine((value) => !value.includes('\0'), { message: 'Invalid characters' })
    .refine((value) => !/[\x01-\x08\x0b\x0c\x0e-\x1f]/.test(value), {
      message: 'Invalid characters',
    });
}
```

## 7. Проверка прав на сервере (RBAC)

Файл `backend/src/services/access.service.ts` (фрагмент). Матрица действий по ролям; при отсутствии права — AppError 403.

```typescript
const rolePermissions: Record<Role, Set<AccessAction>> = {
  OWNER: new Set(['read', 'edit', 'delete', 'manage_access', 'comment', 'websocket_edit']),
  EDITOR: new Set(['read', 'edit', 'comment', 'websocket_edit']),
  COMMENTATOR: new Set(['read', 'comment']),
  READER: new Set(['read']),
};

async requireAction(userId, noteId, action) {
  const role = await this.getUserRole(userId, noteId);
  if (!role || !rolePermissions[role].has(action)) {
    throw new AppError(403, 'Insufficient permissions');
  }
  return role;
}
```
